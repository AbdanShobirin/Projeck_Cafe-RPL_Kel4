## Run Locally

Clone the project

```bash
  git clone https://github.com/AbdanShobirin/Projeck_Cafe-RPL_Kel4
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
